# Project Description

## Brainware Analysis for Early ASD Detection

This project focuses on detecting Autism Spectrum Disorder (ASD) using EEG brainwave signals and machine learning techniques.

The dataset contains EEG frequency band features collected from subjects. These features include delta, theta, alpha, beta, and gamma brainwave bands measured from left and right frontal brain regions.

Machine learning techniques are applied to analyze these EEG signals and identify patterns that differentiate ASD and non-ASD subjects.

## Objectives

- Detect ASD using EEG brainwave signals
- Identify important EEG biomarkers
- Build a machine learning prediction model
- Deploy the model for ASD risk prediction

## Methodology

The project follows a machine learning pipeline:

1. Data preprocessing
2. Feature engineering
3. Data balancing using SMOTE
4. Model training using XGBoost
5. Model evaluation
6. Biomarker identification using SHAP
7. Deployment for prediction

The trained model predicts whether a subject shows ASD patterns based on EEG features.
